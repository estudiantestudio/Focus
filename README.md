# Focus C.E.R.O.

Aplicación web personal para organizar:

- colegio;
- IB;
- PAES;
- tareas;
- hábitos;
- sesiones de estudio;
- planificación semanal;
- método C.E.R.O.;
- progreso académico.

## Archivos principales

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `service-worker.js`

## Carpeta de iconos

- `icons/icon-192.svg`
- `icons/icon-512.svg`

## Publicar en GitHub Pages

1. Crear un repositorio público.
2. Subir todos los archivos.
3. Entrar a Settings.
4. Entrar a Pages.
5. Seleccionar Deploy from a branch.
6. Seleccionar main.
7. Seleccionar /root.
8. Guardar.

## Instalar en iPhone

1. Abrir la aplicación en Safari.
2. Presionar Compartir.
3. Seleccionar Agregar a pantalla de inicio.
4. Presionar Agregar.
